## Main concepts

- Declarative data fetching
- Schema Definition Language (SDL) with its type system
- Getting the data with `Query`
- Writing the data with `Mutation`
- Realtime updates with `Subscription`

```notes
- Client can specify exactly what data it needs
```

### Your server data = Graph

```notes
- Think of your server data as a graph
- It has nodes, types/objects, connections
- You can retrieve data with traversing the graph
```
